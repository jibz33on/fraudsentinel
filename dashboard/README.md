# Dashboard — Stitch Import Guide

## How to import Stitch export:
- Copy all .html files → dashboard/templates/
- Copy styles.css → dashboard/static/css/
- Copy any .js files → dashboard/static/js/
- Copy icons/images → dashboard/static/assets/

## File naming:
- Main dashboard → templates/index.html
- Transaction detail → templates/transaction.html
- User profile → templates/user.html
